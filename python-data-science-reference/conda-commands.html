Creating Environments Activating Environments Installing Libraries Updating Conda Launching Spyder & Jupyter

Creating Conda Environments

In the terminal we use the conda create command to create a new conda python 3.6 environment called forecast_model:

conda create --name forecast_model python=3.6

Activating Conda Environments

On Windows we enter the following to activate the forecast_model environment:

activate forecast_model

On Linux and Mac we enter the following to activate the forecast_model environment:

source activate forecast_model

Installing Libraries in Conda Environment

In the terminal, with the required environment activated, we use the conda install command to install the Seaborn library:

conda install seaborn

Updating Conda

conda update conda

Update Seaborn library in the active environment:

conda update seaborn

Launching Jupyter & Spyder from Conda Environment

Activate the required environment and then launch jupyter notebook using the following command:

jupyter-notebook

Activate the required environment and then launch the Spyder IDE using the following command:

spyder
